# Modelo para asciidoctor-pdf

## Instalación asciidoctor-pdf

Para poder empregar este modelo é preciso instalar `asciidoctor-pdf`. 

```bash
gem install asciidoctor-pdf --pre
gem install rouge
```

## Como usar

```bash
# Clonar repo
git clone https://github.com/vifito/asciidoc-template

# ... escribir en documento book.adoc

# Xerar PDF
make

# Abrir PDF con evince
make view
```
